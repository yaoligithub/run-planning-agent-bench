# Coach Research Notes (for AI Decision)

## Core policy (must follow)
1. Injury first: any clear injury signal => downgrade intensity.
2. 80/20 intensity distribution over rolling windows.
3. No back-to-back hard sessions.
4. Long run control: mostly easy; no long threshold blocks.
5. Prefer consistency over hero workouts.

## Practical heuristics
- If soreness >= 7 or multi-day soreness trend: rest/recovery.
- If fatigue trend rises for multiple days: reduce load before quality.
- Morning unrecovered + planned hard day: convert to easy/recovery/rest.
- Post-run high strain should affect next-day schedule.

## Output style
- Keep recommendation concrete and simple.
- Provide short rationale bullets.
